Leccion 1

Explicacion de la importancia semantica en html5, lo primero explicado es que las etiquetas <h1>, <h2>, ......., <hn> son simplemente para que el codigo de html sea mas legible para el programador, y agregar una estructura jerarquica a los encabezados nada mas, ya que aunque si haya una diferencia de tamaño inicialmente todo eso se arregla facil en css pudiendole poner el tamaño que desees a la imprenta.

Otras etiquetas mostradas son:

  • <header> : utilizada para mostrar un encabezado, logo de la marca, lema, informacion incluso navegacion.
  • <nav> : utilizada para algun contenido referido a la navegacion dentro de nuestro sitio web
  • <section> : utilizada para dar orden a la pagina dividiendola en secciones para que sea mas legible al programador.
  • <article> : uilizada para indicar un articulo, generalmente van dentro de alguna seccion pero no es obligatorio.
  • <aside> : utilizada para comunicar que hay algo en ese elemento relacionado con el contenido de la pagina principal.
  • <footer> : utilizada para poner informacion en el pie de pagina
es cierto que estas etiquetas no son 100% necesarias pero le dan mucha legibilidad a la estructura del codigo.

Leccion 2

En esta leccion vemos el tema de las listas acerca de como es un elemento muy comun ya que se usa frecuentemente, hay dos tipos de listas: las listas desordenadas y las listas ordenadas.
Las listas desordenadas utilizan la etiqueta <ul>, y para cada uno de los elementos internos de la lista desordenada se utiliza <li>, ademas se puede crear listas dentro de listas si se quisiera implementando el mismo formato.

Ejempo de lista desordenada:

<ul> (etiqueta lista desordenada)
     <li>cafe</li>
     <li>tortitas</li>
     <li>medialunas</li>
     <li>galletas
        <ul>
          <li>de chocolate</li>
          <li>dulces</li>
          <li>de agua</li>
        </ul>
     </li>
     <li>manteca</li>
</ul>

En cambio las listas ordenadas estan hechas con el proposito de dar pasos a seguir, dar un orden, su etiqueta es <ol> y al igual que las listas desordenadas para cada uno de sus elementos se utiliza la etiqueta <li> y se pueden crear listas dentro de otras.

Ejemplo de lista ordenada: como armar una oreo doble

<ol> (etiqueta lista ordenada)
     <li>abre el paquete</li>
     <li>saca la galleta</li>
     <li>has una doble oreo</li>
        <ol>
          <li>despegar la parte superior</li>
          <li>poner otra galleta en el medio</li>
          <li>poner de nuevo la parte superior</li>
        </ol>
     </li>
     <li>Disfruta!!!</li>
</ol>

Leccion 3

Un problema comun es que html utiliza ciertos caracteres para su sintaxis, necesitamos una forma de diferenciar entree esos caracteres como html y los mismo caracteres que el contenido, es decir, que necesitamos una forma que el navegador no interprete esos caracteres como de html.
Para eso se utilizan referencias de entidades de caracteres, hay tres caracteres que siempre tiene que ser evitados para no tener malas interpretaciones estos son: < (su equivalente es '&lt;'), > (su equivalente es '&gt;') y el & (su equivalente '&amp;'), hay otro como el caso de '&nbsp;' que se utiliza para dar mas espacios a una oracion o bien para que palabras no se separen, y siempre queden juntas sin importar que sean palabras distintas, '&copy;' es el simbolo de copyrigth (©), '&quot;' es el simbolo de las comillas ("). Estos son solo algunos ejemplos de referencias de entidades de caracteres.

Leccion 4

Enumeremos tres formas de enlazar o linkear:

  1. Los vinculos se crean con un atributo href (referencia de hypertexto) el valor de dicho elemento puede ser relativo o absulto, estos links son creados para archivos desntro del mismo directorio y por eso no hace especificar mucho del enlace. Siempre es bueno especificar un atriburo title a la etiqueta 'a', dicho atrubito es itulizado por los lectores de pantalla para ayudarlos a ver mejor la pagina, el contenido entre la apertura y el cierre de las etiquetas es el contenido que va a ver en la pagina web cuando veas el link, en cambio si usamos una etiqueta <DIV> el contenido que este ahi (ya sea una imagen, un logo, etc) sera el que tendra el link.

    Ej: <a href='una direccion.html' title='misma direcc link'>Este link lleva al archivo en el mismo directorio</a>
    Ej (con etiqueta DIV): <a href='una direccion.html' title='misma direcc link'
      <div>Aqui adentro se puede poner cualquier cosa y se hace un link en el mismo directorio</div>
    </a>

  2. Tambien existen los enlaces externos que que a diferencia de los anteriores estos llevan el 'http://' por que generalmente los enlaces externos stan alojados en un nombre de dominio diferente a nuestro sitio web (vease un link a youtube, fb, etc). Hay una caracteristica muy buena a destacar por lo util que resulta y es el atributo 'target'. Cuando el atributo target tiene el valor 'target=_blank' obliga al navegador a abrir esta pagina en una nueva pestaña o una nueva ventana, esto se realiza con la intencion de que la gente no abandone nuestra pagina del todo.

    Ej: <a href='https://www.youtube.com/watch?v=35XFAkwmU4c' target='_blanc' title='Ve eeste video!!!'>video de you tube</a>
    Este seria el resultado: video de you tube

  3. El ultimo tipo de enlace es el utilizado en esta pagina y sirve para conectar mas facilmente las secciones de un apagina web y que sea de mas facil acceso el contenido este tipo de paginas es muy utilizada ultimamente.

    Ej:
    <a href='#Leccion1'>#Leccion1</a>

    en este caso le tendriamos que dar un 'id' a la seccion para que quede linkeada
    <section id='section1'>

En esta ultima leccion de html veremos como cargar una imagen, las imagenes se cargan atravez de la etiqueta:
<img> src='imagen aqui' width='400' height='235' alt='imagen con comilla'>
Las imagenes pueden ser subidas por directorio o por enlace externo por el mismo metodo solo que se le agregaria el 'http://' en 'src' width y height son las especificaciones de la imagen no son necesarias pero dejarlas aclaradas resulta beneficioso en varios casos (como por ej que la pagina se reserva el lugar de la imagen y asi la pagina no andara dando saltos de texto por imagenes que no carguen) por ultimop 'alt' ayuda a la gente con impedimentos visuales
