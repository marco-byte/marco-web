Leccion 1

Explicacion de la importancia semantica en html5, lo primero explicado es que las etiquetas <h1>, <h2>, ......., <hn> son simplemente para que el codigo de html sea mas legible para el programador, y agrgar una estructura jerarquica a los encabezados nada mas, ya que aunque si haya una diferencia de tamaño inicialmente todo eso se arregla facil en css pudiendole poner el tamaño que desees a la imprenta.

Otras etiquetas mostradas son:

  • <header> : utilizada para mostrar un encabezado, logo de la marca, lema, informacion incluso navegacion.
  • <nav> : utilizada para algun contenido referido a la navegacion dentro de nuestro sitio web
  • <section> : utilizada para dar orden a la pagina dividiendola en secciones para que sea mas legible al programador.
  • <article> : uilizada para indicar un articulo, generalmente van dentro de alguna seccion pero no es obligatorio.
  • <aside> : utilizada para comunicar que hay algo en ese elemento relacionado con el contenido de la pagina principal.
  • <footer> : utilizada para poner informacion en el pie de pagina
es cierto que estas etiquetas no son 100% necesarias pero le dan mucha legibilidad a la estructura del codigo.
